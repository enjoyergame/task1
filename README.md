Консольная утилита на языке C для замены произвольных последовательностей байтов в файлах.

1. **Строго ограниченный буфер памяти:** Программа считывает данные из входного файла блоками фиксированного размера $N$ байт (задается при компиляции через CMake). В памяти одновременно находится не более $N$ байт из файла.
2. **Отсутствие возврата по файлу:** Каждый байт читается более одного раза. Алгоритм не использует перемещение указателя чтения назад.
3. **Отсутствие предобработки шаблонов:** Алгоритм не использует предварительно вычисленные таблицы сдвигов или переходов.
4. **Динамический рекурсивный откат:** При частичных совпадениях шаблона реализован конечный автомат с рекурсивным откатом. В момент срыва совпадения программа выталкивает первый придержанный символ в выходной файл, а остальные перепроверяет рекурсивно.
5. **Поддержка любых бинарных данных:** Программа корректно обрабатывает любые последовательности байтов, включая нулевые байты (`\x00`). Для безопасной передачи спецсимволов через аргументы командной строки поддерживается HEX-формат.
6. **Произвольный размер шаблона поиска:** Искомая последовательность может иметь любой размер, в том числе превышающий размер буфера.

## Структура проекта

* `lib/include/replacer.h` — Публичный интерфейс библиотеки.
* `lib/src/replacer.c` — Реализация конечного автомата с рекурсивным откатом.
* `lib/src/utils.h` / `utils.c` — Приватные утилиты библиотеки для валидации и парсинга HEX-последовательностей.
* `src/main.c` — Главное консольное приложение
* `cmake/Testing.cmake` — Модуль для автоматического скачивания фреймворка Unity и регистрации тестов.
* `tests/replacer_tests/test_replacer.c` — Тесты.

## Сборка проекта

Сборка осуществляется с помощью системы **CMake**.

1. Сгенерировать файлы конфигурации CMake (размер буфера чтения по умолчанию $N = 1024$):
   cmake -B build
   cmake --build build

## Использование программы
.\build\src\app.exe input.txt output.txt "hello" "world"

.\build\src\app.exe input.bin output.bin "0xDEADBEEF" "0xCAFEBABE0001"

## Тесты

cmake --build build --target replacer_tests

cd build
ctest --verbose
